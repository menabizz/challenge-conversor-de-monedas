# Challenge del Conversor de Monedas

¡Bienvenido! A través de este código, podrás realizar una conversión automática de distintas monedas. Y al final podrás ver un historial de tus conversiones.

Es muy sencillo, solo necesitas seguir los siguientes pasos:
* Descarga el código y abrelo en tu aplicación de confianza ( ¡yo utilizo IntelliJ IDEA! )
* En la clase "AppParaElUsuario" inicia un Run para probar el código.

  ![imagen](https://i.imgur.com/crMKSe6.png)  

* A continuación se nos mostrará un menú como el siguiente, donde podremos escoger la opción que queramos

  ![imagen](https://i.imgur.com/9xb7hzx.png)

* Una vez elegida la opción, podremos ingresar el monto que queremos convertir. ¡El programa hará el resto!
  
   ![imagen](https://i.imgur.com/58Gk1XR.png)  

* Puedes ir realizando distintas conversiones. Cuando quieras finalizar, solo debes apretar la opción de salir del programa.

 ### ¡Y listo!
¡Te mostrará una lista con todas tus conversiones realizadas!

   ![imagen](https://i.imgur.com/djry6ZR.png)
